HOW TO BUY ICONS

This directory is for icons used in the "How to Buy" section to represent each step of the purchase process.

Recommended specifications:
- File format: PNG (as specified)
- Dimensions: 40x40 pixels
- Transparent background
- Style: Simple, clear iconography

Expected filenames:
- icon1.png (Swap icon)
- icon2.png (Download icon)
- icon3.png (Copy/Swap icon)

These icons appear in the circular containers in the "How to Buy" section. You can replace them with your own icons while maintaining the same filenames, or update the HTML to reference your custom filenames.

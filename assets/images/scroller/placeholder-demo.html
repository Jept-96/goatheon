Instructions

To use the infinite image scroller, you need to create 6 PNG images with the following specifications:

  • Format: PNG
  • Dimensions: 300x200 pixels
  • Filenames: placeholder1.png, placeholder2.png, etc.

Place these images in the assets/images/scroller/ directory.

If you want to use different filenames or dimensions, you'll need to update the HTML and CSS accordingly.

HTML Structure

<div class="scroller-container">
    <div class="scroller-content">
        <div class="scroller-item"><img src="assets/images/scroller/placeholder1.png" alt="Gallery Image 1"></div>
        <div class="scroller-item"><img src="assets/images/scroller/placeholder2.png" alt="Gallery Image 2"></div>
